Project Overview

Project Overview

This project is a web application for a Middle Eastern clothing boutique called Mehr Shop. It allows users to explore cultural fashion, view clothing items, and interact with products online.

Purpose

The purpose of this website is to promote Middle Eastern fashion and provide users with an easy and engaging online shopping experience.

Users

  • Online shoppers looking for cultural clothing
  • People interested in Middle Eastern fashion
  • Customers searching for traditional and modern outfits
  • General visitors exploring fashion styles

Website Content

The website includes product images, prices, clothing categories, an about page, a contact form, and a project documentation page.

Website Pages

This website includes five main pages:

  • Home (index.html)
  • About Page
  • Gallery Page
  • Contact Page
  • Project Overview Page

All pages are connected through a navigation bar for easy access.

Dynamic Functionality

The Mehr Shop website includes JavaScript and jQuery-based interactivity to improve user experience.

1. Add to Cart System

Users can click "Add to Cart" buttons to select products. Items are stored using JavaScript arrays and confirmation messages appear.

2. Product Filtering

Users can filter clothing items by category such as dresses, abayas, or casual wear.

3. Form Validation

The contact form checks required fields (name, email, message) before submission. Errors are shown if fields are empty.

4. Image Gallery Interaction

Clicking on images opens a larger preview using a modal or jQuery UI dialog.

5. jQuery UI Features

The site uses jQuery UI for dialogs, popups, and interactive elements.

6. AJAX (Optional)

AJAX may be used to load product data without refreshing the page.
