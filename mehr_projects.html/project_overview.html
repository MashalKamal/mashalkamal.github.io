Project Overview

Project Overview

This project is a web application for a Middle Eastern clothing boutique. It allows users to explore clothing styles and learn about cultural fashion.

Purpose

The purpose of this website is to promote Middle Eastern fashion and provide users with an easy and engaging browsing experience.

Users

Target users include online shoppers, fashion enthusiasts, and individuals interested in cultural clothing and traditions.

Website Content

The website includes Home, Gallery, About, Contact, and Project Overview pages.

Page Design

Home Page
Purpose: Introduce website
Users: All visitors
Content: Featured outfits, intro text
Inputs: None
Actions: Navigate to gallery

Gallery Page
Purpose: Display products
Users: Shoppers
Content: Product images and descriptions
Inputs: Buttons (Add to Cart)
Actions: Alert message when clicked

About Page
Purpose: Provide brand information
Users: Visitors
Content: Story and mission
Inputs: None
Actions: Read-only

Contact Page
Purpose: Collect user feedback
Users: Visitors
Content: Form (name, message)
Inputs: Text fields
Validation: Required fields
Actions: Submit form

Project Overview Page
Purpose: Show project requirements
Users: Instructor
Content: Documentation
Inputs: None
Actions: Read-only

Dynamic Functionality

JavaScript is used for interactive features such as the Add to Cart button and form validation in the contact page.
